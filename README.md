# モダンウェブサイト

## 概要
最適化されたReactベースのモダンなウェブサイトプロジェクトです。パフォーマンスとメンテナンス性に優れたコード構造を採用しています。

## 特徴
- Reactコンポーネントベースの構造
- モジュール化されたCSS
- 最適化されたJavaScript
- パフォーマンス最適化
- アニメーションとインタラクション
- レスポンシブデザイン

## ファイル構造
```
/
├── public/              # 静的ファイル
│   ├── assets/          # 画像、フォントなどのアセット
│   └── ...
├── src/                 # ソースコード
│   ├── components/      # Reactコンポーネント
│   ├── js/              # JavaScriptモジュール
│   │   ├── animations.js  # アニメーション関連
│   │   ├── ui.js          # UIインタラクション
│   │   └── main.js        # メインJSファイル
│   ├── styles/          # CSSスタイル
│   │   ├── components/    # コンポーネント別CSS
│   │   ├── animations.css # アニメーション
│   │   ├── layout.css     # レイアウト
│   │   ├── main.css       # メインスタイル
│   │   └── index.css      # スタイルのエントリーポイント
│   ├── pages/           # ページコンポーネント
│   └── index.jsx        # アプリケーションのエントリーポイント
├── package.json         # 依存関係と設定
├── vite.config.js       # Vite設定
└── ...
```

## 使い方

### 開発環境のセットアップ
```
npm install
```

### 開発サーバーの起動
```
npm run dev
```

### ビルド
```
npm run build
```

### プレビュー
```
npm run preview
```

## 最適化について
- CSSファイルは機能ごとに分割されています
- JavaScriptモジュールは関連する機能ごとに整理されています
- 画像の遅延読み込みを実装
- アニメーションはパフォーマンスを考慮して最適化
- コードを分割してバンドルサイズを最小化
